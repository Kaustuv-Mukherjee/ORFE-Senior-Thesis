# ORFE-Senior-Thesis
This public repository contains all the code executed for my Operations Research and Financial Engineering Senior Thesis: Modeling CitiBike Mobility as a Directed Graph with Magnetic Laplacian-Based Graph Neural Networks 

The open source CitiBike Trip Histories dataset I used is linked here: https://s3.amazonaws.com/tripdata/index.html

The dataset contains zip files of Excel spreadsheets, which each contain monthly trip-level data.
